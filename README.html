Inkscape2Symbol Help

In Inkscape...

  1. Create a simple shape. All measurement units are supported, but 'pixels' is preferred. Single-part shapes are better, but islands/lakes are supported. Multi-shape drawings are supported, but individual styles will be lost. The shape must not be in a group. Clear any transformations if you can before saving.
  2. Set the document size to the size of the shape (Ctrl+Shift+D for document properties).
  3. Save as Plain SVG to your destination of choice.

In QGIS...

  1. Select the input .svg file that you created in Inkscape
  2. Select the output .svg to your destination of choice. It is recommended that you choose the default QGIS User SVG folder (see Settings → Options → System → SVG Paths)
  3. Set fill and outline colors for the default style. Notice the output preview update as the changes are made.
  4. Click OK to save the symbol
